# Sementes Digitais

![Logo do Sementes Digitais](assets/branding/logo_sementes_digitais.png)

**Sementes Digitais — cultivando oportunidades, colhendo futuros.**

Repositório-base do eixo de **Matemática** do cursinho popular **Sementes Digitais**, organizado para uso no GitHub e para espelhamento no Google Drive quando necessário.

## Propósito

Este repositório foi estruturado para servir como base viva de produção pedagógica. A ideia é manter no GitHub o núcleo de planejamento, atualização, versionamento e registro dos materiais, enquanto PDFs, impressos e documentos finais podem ser espelhados em outra plataforma quando a equipe julgar necessário.

## O que já está pronto neste pacote

- estrutura-base do repositório;
- README principal com visão institucional;
- eixo completo de **Matemática**;
- ementa, objetivos e organização pedagógica;
- conteúdo programático em 8 módulos;
- cronograma completo de **32 semanas**;
- planos semanais detalhados;
- cartilhas do estudante;
- listas de exercícios e gabaritos;
- avaliações diagnósticas, simulados e revisão final;
- modelos para replicação nos demais eixos.

## Princípios pedagógicos do eixo

1. foco em competências e habilidades;
2. leitura crítica de situações reais;
3. resolução de problemas com sentido social;
4. prática intensiva no estilo Enem;
5. revisão por erros e devolutiva formativa;
6. materiais simples, claros e reaproveitáveis.

## Como usar na prática

1. Leia `matematica/README.md`.
2. Comece pela pasta `01-visao-geral`.
3. Siga para `02-conteudo-programatico` para entender a lógica do percurso.
4. Use `03-planos-semanais` durante a execução do curso.
5. Distribua aos estudantes as cartilhas e listas.
6. Registre adaptações e ajustes ao longo da aplicação.

## Observação importante

A **sequência semanal e modular** presente neste repositório é uma **organização pedagógica do Sementes Digitais**, construída a partir de documentos oficiais de referência, mas adaptada ao contexto do cursinho, à carga horária de 2h/semana de Matemática e ao foco em permanência, progressão e preparação para o Enem.
