# Planos semanais — Ciências Humanas

Esta pasta reúne os planos de aula semanais do eixo de Ciências Humanas, organizados em 8 meses de execução, com 4 semanas em cada mês.

Cada plano contém identificação, objetivo, conteúdos, habilidades, sequência didática, missão semanal, produto esperado e campo de observações pós-aula.
