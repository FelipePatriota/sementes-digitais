# Planos semanais de Redação

Os planos semanais estão organizados em **8 meses**, com **4 semanas por mês**, totalizando **32 semanas**. Cada arquivo foi pensado para uso prático da equipe: objetivo da aula, conteúdos, competências mobilizadas, sequência sugerida, missão semanal e campo de observações pós-aula.
