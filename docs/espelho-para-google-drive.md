# Espelho opcional para Google Drive

Embora o GitHub seja o núcleo de produção deste eixo, vocês podem manter um espelho simplificado no Google Drive com a seguinte lógica:

- **Mês**
  - **Matemática**
    - **Semana 01**
    - **Semana 02**
    - **Semana 03**
    - **Semana 04**

## O que vai para o Drive

- PDFs finais;
- cartilhas para impressão;
- listas de exercícios;
- gabaritos para uso interno;
- materiais de aula que precisam circular rapidamente.

## O que deve ficar prioritariamente no GitHub

- ementa;
- conteúdo programático;
- plano semanal em edição;
- histórico de alterações;
- observações da equipe;
- estrutura do eixo.
